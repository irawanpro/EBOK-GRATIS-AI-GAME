Materi 2: Struktur Game

Sekarang, bayangkan game itu seperti rumah. Rumah tanpa struktur? Hancur. Game juga sama, ada komponennya:

UI (User Interface)

Antarmuka yang dilihat dan digunakan pemain. Contoh: tombol, menu, skor, timer.

Game Area

Area di mana gameplay terjadi. Contoh: papan puzzle, layar permainan.

Game Logic

Aturan dan sistem di balik layar. Contoh: cara skor dihitung, bagaimana objek bergerak.

Feedback System

Sistem yang memberitahu pemain apa yang terjadi. Contoh: pesan "Selamat!", efek suara, animasi.

Tanpa struktur ini, game kita kayak jalan tanpa petaâ€”nggak ada arah.
